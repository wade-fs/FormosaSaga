  ### 核心機制

  好感度層級（累積解鎖）：

   層級                          | 門檻                         | NPC 行為
  -------------------------------|------------------------------|------------------------------
   陌生                          | 0                            | 客套應對，只談公開話題
   熟識                          | 25                           | 認得你了，多聊幾句日常
   朋友                          | 50                           | 分享個人故事，給予任務線索
   信任                          | 75                           | 拜託私事，給予特殊協助與物品
   知己                          | 90                           | 吐露心聲，解鎖最深層歷史記憶

  好感度增長途徑：

  •  greet <NPC>  打招呼 → +2
  •  ask <NPC> <話題>  對話成功 → +1
  • 完成 YAML 中  related_npc  關聯任務 → +15

  YAML 設計（ relationship_dialogues ）： 內容作者只需在 YAML 中按層級鍵（
  stranger/familiar/friend/trusted/intimate
  ）填入話題與台詞，系統會自動依玩家目前的好感層級「累積解鎖」對應話題，無需更改任何程式碼。
