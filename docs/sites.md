# 名稱永遠從 Era Overlay 查詢。

這樣玩家在 1700 年看到的是：

打貓

1905 年看到的是：

民雄庄

2026 年看到的是：

民雄鄉

但底層永遠都是同一個 Settlement：

id: minxiong

這樣你的世界資料庫才能支撐真正的歷史演進，而不會被現代地名綁死。
