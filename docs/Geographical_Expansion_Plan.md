# 《源流福爾摩沙》小說卷一地理版圖深化與擴展規劃

為了讓民雄之外的聚落（嘉義市、新港、台南安平、鹿港、彰化八卦山）更具地圖感與可玩性，擺脫目前每個聚落僅有少數地標（Sites）的單薄現狀，我們在此規劃了深化的地理版圖、專屬任務（Quests）與職涯（Careers）深度融合機制。

---

## 🗺️ 1. 聚落地圖版圖擴充 (Settlement Layout Extensions)

### 📌 嘉義市聚落 (Chiayi City) — 文化與信仰的中樞
*   **現狀**：僅有主聚落、城隍廟、嘉義車站。
*   **新增地標規劃**：
    1.  `chiayi_east_market` (東市場)：全台最古老的公有市場之一，作為商人 (Merchant) 的核心交易地，提供獨特的小吃調配與古老南北貨交易。
    2.  `chiayi_lam_tian_academy` (藍田書院/嘉義孔廟)：文人 (Scholar) 的集聚地，存放許多前朝文獻，也是發佈地方誌的文化溫床。
    3.  `chiayi_wood_factory` (嘉義製材所)：日治時期「東洋第一製材所」，與阿里山林鐵連通，提供匠人 (Artisan) 獲取珍貴高級檜木踏印的場所。

### 📌 新港鄉 (Singang) — 拓墾與媽祖的香火
*   **現狀**：主聚落、新港庄市集。
*   **新增地標規劃**：
    1.  `singang_fengtian_temple` (奉天宮)：歷史悠久的媽祖廟，與廟委勢力掛鉤，也是早期開墾的精神支柱。
    2.  `singang_boardwalk_field` (笨港開台第一庄)：象徵最早閩南移民涉足開墾的農田，為農人 (Farmer) 提供培育特殊抗旱稻作的實驗基地。
    3.  `singang_ancient_ferry` (笨港古渡口)：連通昔日笨港（新港/北港）水路，為商人的走私或客運路線。

### 📌 台南府城 (Tainan) — 跨越四百年的歷史疊加
*   **現狀**：主聚落、安平、赤崁樓、孔廟、西來庵、西拉雅社。
*   **新增地標規劃**：
    1.  `tainan_da_tianhou_temple` (大天后宮)：昔日寧靖王府，蘊藏明鄭至清領轉折期的宮廷哀歌記憶。
    2.  `tainan_zeelandia_city` (熱蘭遮城內城/安平古堡)：歷史層 v0.2 的核心，荷蘭守軍與鄭成功談判的決策之地。
    3.  `tainan_wu_channel` (五條港遺址)：清領時期的商業港渠，商人的「船貨買賣」秘境，包含水路運送委託。

### 📌 鹿港鎮 (Lukang) — 手藝工匠與郊商的輝煌
*   **現狀**：主聚落、老街、天后宮、龍山寺、民俗館。
*   **新增地標規劃**：
    1.  `lukang_nine_turns_lane` (九曲巷)：防風防盜的獨特防衛性巷道，將實作「迷宮式探索」，玩家需藉由地理理解才能通過。
    2.  `lukang_moka_shop` (鹿港意樓/摸乳巷)：隱藏在老街深處的幽靜景點，作為口述傳說與人情踏印的特殊獲取點。

## 📜 2. 專屬深水任務設計 (Quest Lines)

### 🌾 任務一：笨港開圳委託 (農人與新港)
*   **觸發點**：新港奉天宮旁的墾首。
*   **要求**：農人等級 >= 2，完成新手引導。
*   **內容**：笨港溪氾濫且灌溉水源不足，玩家需與同伴合作勘查笨港古渡口，排除河道碎石，引水灌溉笨港開台第一庄。
*   **獎勵**：解鎖「笨港大圳」踏印、農人修練點、新港聚落凝聚力。

### ⚓ 任務二：鄭成功大員登陸秘辛 (文人與安平)
*   **觸發點**：v0.2 時代的安平（大員海灣）明鄭軍官。
*   **要求**：文人等級 >= 2。
*   **內容**：在鄭成功軍隊登陸前，協助翻譯熱蘭遮城內荷蘭守軍的佈防信件，並向鹿耳門漁民考據潮汐變遷，引導明鄭主力戰船避開淺灘順利登陸。
*   **獎勵**：解鎖「鹿耳門潮湧」傳承記憶碎片、文人修練點。
