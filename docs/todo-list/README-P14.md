  #### 🕐 虛擬時間系統 (schedule_d.c)                                                                  
                                                                                                       
  • 15 秒現實 = 15 遊戲分鐘，遊戲從早上 08:00 開始                                                     
  • 週期廣播  GameTimeTick  事件（含  time_str  +  minutes ）                                          
  •  is_time_in_range()  支援跨日時段（如  "22:00-06:00" ）                                            
  • 玩家可輸入  time  指令查看目前時間與場景 NPC 動態                                                  
                                                                                                       
  #### 🧠 NPC 智慧日程 (npc.c)                                                                         
                                                                                                       
  •  on_time_tick()  接收廣播後判斷應在哪個地點、做什麼事                                              
  •  execute_routine()  自動  move_object  到目標 Site 並廣播移動訊息                                  
  •  look  場景時顯示「這裡有：老站長 正在慢條斯理地打掃月台。」
